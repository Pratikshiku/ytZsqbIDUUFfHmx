## Professional Title Evaluation Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/7a75ccde20494b04bd6b1908d25ddf86/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): Regular users, Reviewers, Administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Review Conditions
5. Forum Information
6. System Announcements
7. Personal Center
8. My Posts
9. My Favorites
10. Results Announcement

**Reviewer Features:**
1. Login
2. Registration
3. Homepage
4. Review Conditions
5. Forum Information
6. System Announcements
7. Personal Center
8. My Posts
9. My Favorites
10. Professional Title Application Review
11. Add Professional Title Application Results
12. View Results Announcement

**Administrator Features:**
1. Login
2. Personal Center
3. User Management
4. Reviewer Management
5. Province Management
6. Review Conditions Management
7. Professional Title Application Management
8. Results Announcement Management
9. Forum Management
10. Customer Service Management
11. Banner Management
12. System Announcements
